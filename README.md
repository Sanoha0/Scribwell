# Smart Writing Book

This is your custom, locally-run AI writing assistant! It features a real-time auto-reading assistant with a debounced sidebar feed.

## How to use:
1. Extract this ZIP file into a folder.
2. Double click the `index.html` file to open it in your web browser (Chrome, Edge, Safari, Firefox, etc.).
3. Enter your OpenRouter API key at the top and click "Save Key".
4. Start writing! When you pause for 3 seconds, the AI will automatically analyze your text.

## Uploading to GitHub
You can upload this entire folder to a GitHub repository. If you turn on GitHub Pages in the repository settings, you'll have a live web app hosted for free. Your API key will stay safe because it uses `localStorage` directly in your browser rather than being uploaded with the files.

# The Master Writing Book (With Auto-Lore & Enter-to-Send)

This version is the ultimate serverless upgrade! It includes everything you requested:

## Features Inside:
1. **Enter to Send:** You can press the Enter key in the prompt box to send your message instantly.
2. **Auto-Filling Story Bible:** Ask the AI to "generate a character profile" or "write worldbuilding rules". It will format the text perfectly, and the Javascript will automatically extract it and inject it straight into your left sidebar's Story Bible boxes!
3. **Default Model:** It is configured to use `openai/gpt-4o-mini` by default (it's fast, incredibly cheap for rapid real-time typing analysis, and very smart). You can change this anytime by opening `app.js` and changing the model name.

## How to use on GitHub Pages:
Upload `index.html`, `style.css`, and `app.js` into your GitHub repository and turn on GitHub Pages.

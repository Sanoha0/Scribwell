# The Master Writing Book (Static & Full-Featured)

This version is the ultimate serverless upgrade! It introduces everything you need for complex worldbuilding and manuscript management without needing a Python backend. It will work flawlessly on GitHub Pages.

## New Features Added:
1. **Chapter Management:** A left sidebar allows you to add, switch between, and delete multiple chapters. Everything auto-saves to your browser.
2. **Story Bible (AI Memory):** A dedicated tab in the left sidebar for your Character Lore and Worldbuilding Rules. Whenever the AI reads your story, it *secretly* reads your Story Bible first, so it remembers who Sanoha Knight is and how your world works!
3. **Export to Word/Google Docs:** A button at the top instantly stitches all your chapters together and downloads a `.doc` file that can be opened natively in Microsoft Word or uploaded directly to Google Docs.
4. **Live Word Count:** Tracks your progress at the bottom of the editor.

## How to use on GitHub Pages:
1. Create a repository on GitHub.
2. Upload `index.html`, `style.css`, and `app.js` into the repository.
3. Turn on GitHub Pages via your repository settings.
4. Access your live app, input your OpenRouter key, and write away! (Your chapters, lore, and API key are saved securely to the `localStorage` of whichever browser/device you are typing on).
